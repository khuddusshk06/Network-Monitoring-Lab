# Network Monitoring & Troubleshooting Lab

## Objective
Monitor network activity and troubleshoot common network issues.

## Tools Used
- Linux
- ping
- traceroute
- nslookup
- tcpdump
- ss

## Tasks Performed

### Network Information Gathering
Commands:
ip a
ip route

### Connectivity Testing
ping 8.8.8.8
ping google.com

### DNS Troubleshooting
nslookup google.com

### Route Analysis
traceroute google.com

### Packet Analysis
tcpdump

## Findings
- Verified internet connectivity
- Checked DNS functionality
- Monitored active connections
- Captured network packets

## Skills Demonstrated
- Network Monitoring
- Linux Administration
- TCP/IP
- DNS Troubleshooting
- Packet Analysis
